# Homebrew Tap

## Install Formula

`brew install florianilch/tap/<formula>`

Or `brew tap florianilch/tap` and then `brew install <formula>`.

## Install Cask

`brew install --cask florianilch/tap/<cask>`

Or `brew tap florianilch/tap` and then `brew install --cask <cask>`.

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
